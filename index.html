POCKETLAB · QUICK GUIDE

Что это. Мини лаборатория для HTML/JS скетчей. Вставляешь код сверху, нажимаешь RUN, смотришь результат снизу.

Базовый сценарий.

  • OPEN — загрузить .html / .txt файл с устройства
  • SAVE — сохранить текущий код в .html
  • PASTE / COPY / CLEAR — работа с буфером и очистка
  • DEMO — случайная демо-страница из папки /demo
  • RUN — запускает или перезапускает превью
  • FULL — полноэкранное превью, выход по квадратику в углу
  • OPEN PAGE — открыть результат в новой вкладке
  • EXPORT APP — сохранить текущий код как отдельный .html без интерфейса PocketLab

Поиск. Вводишь строку в поле search in code, жмешь SEARCH или Enter.

HTML режим. Если вставлен полный документ с <html>, он запускается как есть. Если фрагмент, PocketLab оборачивает его в минимальный HTML каркас.
